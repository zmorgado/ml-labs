# AI Specialization - Module 2 (Machine Learning)

This folder contains Jupyter notebooks for the second module of the AI specialization, focused on machine learning fundamentals and classification workflows.

## Notebooks

- `01_classification_intro.ipynb` - Introductory classification concepts and baseline models.
- `01_regression_exercise.ipynb` - Regression practice notebook.
- `02_classification_exercise.ipynb` - Core classification exercise.
- `02_classification_exercise_diabetes.ipynb` - Classification exercise on a diabetes dataset (includes PCA/model comparison).
- `03_classification_advanced.ipynb` - Advanced classification techniques and tuning.
- `04_data_standardization.ipynb` - Feature scaling and standardization concepts.
- `09_feature_selection_theory.ipynb` - Theory and examples for feature selection.
- `t4_data_structures.ipynb` - Supporting notebook for data structures review.

## Notes

- Notebook names were normalized to be consistent, readable, and shell-friendly.
- An exact duplicate notebook was removed during cleanup.

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Launch Jupyter:

```bash
jupyter notebook
```
