# AI Specialization - Module 2 (Machine Learning)

This module is organized into **theory** and **exercises** notebooks.

## Structure

### Theory
- `theory/01_classification_intro.ipynb` - Introductory classification concepts and baseline models.
- `theory/03_classification_advanced.ipynb` - Advanced classification techniques and tuning.
- `theory/04_data_standardization.ipynb` - Feature scaling and standardization concepts.
- `theory/09_feature_selection_theory.ipynb` - Feature selection concepts and examples.
- `theory/t4_data_structures.ipynb` - Supporting data structures review.

### Exercises
- `exercises/01_regression_exercise.ipynb` - Regression hands-on practice.
- `exercises/02_classification_exercise.ipynb` - Core classification exercise.
- `exercises/02_classification_exercise_diabetes.ipynb` - Classification exercise on a diabetes dataset.

## Setup

1. Create and activate a Python virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:

```bash
jupyter notebook
```
